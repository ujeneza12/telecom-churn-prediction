# Telecom Churn Prediction

A prediction system that identifies high-risk telecom customers, calculates churn probability, and suggests retention strategies.

## Tech Stack
Python · Scikit-learn · Pandas · Data Analysis

## Structure
- `data_loader.py` — loads and preprocesses the dataset
- `churn_model.py` — trains the prediction model
- `analysis.py` — exploratory data analysis
- `metrics.py` — evaluates model performance
- `segmentation.py` — customer segmentation logic
- `main.py` — entry point

## Run
```bash
pip install -r requirements.txt
python main.py
```
